# Image Prompts

## Visual Families

- Hero art: premium temple-archive cinematic image, rich but readable.
- Character sheet: front/profile/symbols, labeled only outside final art if needed.
- Story scene: one clear narrative moment.
- Coloring page: clean black line art, large colorable shapes, no dense shading.
- Booklet cover: premium hardbound feel, title space, ornamental border.
- Talapatra card: bamboo/palm-leaf strip, sparse text, sacred-object feel.

## Coloring Prompt Rules

Ask for:

- White background.
- Thick clean black outlines.
- Large open areas.
- Child-friendly faces.
- No tiny text.
- No dense cross-hatching.
- Respectful deity iconography.

Avoid:

- rough diagrams,
- stick figures,
- crowded faces,
- shaded grayscale,
- unreadable ornaments,
- accidental extra limbs or weapons.
