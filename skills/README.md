# Internal Operating Skills

These are repo-level skill blueprints for running the platform consistently. They can later be converted into Codex skills or scheduled automation prompts.

## `source-librarian`

Responsibilities:

- find authentic sources,
- validate usage rights,
- capture source hierarchy,
- mark source confidence,
- prevent copyrighted paraphrase pipelines.

Output:

- source records,
- source section records,
- source brief for each story.

## `entity-linker`

Responsibilities:

- extract entities,
- create lightweight entity nodes,
- create typed edges,
- attach provenance,
- flag weak claims.

Output:

- `kg_nodes`,
- `kg_edges`,
- related story/entity trails.

## `cinematic-reteller`

Responsibilities:

- write cinematic narratives from source briefs,
- preserve devotional dignity,
- keep unsupported invention out,
- produce adult and audio-first versions.

Output:

- adult narrative,
- chapter plan,
- hook,
- voiceover draft.

## `kids-adapter`

Responsibilities:

- create age-banded child versions,
- simplify without trivializing,
- add drawing prompts,
- add parent conversation prompts.

Output:

- 5-8 version,
- 9-12 version,
- activity pack.

## `seo-aeo-publisher`

Responsibilities:

- produce direct answers,
- meta title/description,
- FAQ,
- schema.org JSON-LD,
- internal links from KG.

Output:

- publish-ready page metadata.

## `digital-book-designer`

Responsibilities:

- convert story packs into premium booklet structure,
- create talapatra inserts,
- define image requirements,
- define audio player chapters.

Output:

- booklet JSON,
- art prompt list,
- audio chapter markers.

## `community-growth-operator`

Responsibilities:

- define unlocks,
- competition loops,
- referral rewards,
- kit CTAs,
- lifecycle email triggers.

Output:

- growth campaign spec,
- lead tags,
- event tracking plan.

## `temple-guide-researcher`

Responsibilities:

- build state/city/place-wise temple guides,
- collect official timings and practical travel data,
- summarize review/vlog patterns without copying,
- capture story, sthala purana, rituals, festivals, and local etiquette,
- mark freshness and verification status.

Output:

- temple profile,
- travel guide,
- source list,
- freshness queue,
- user-tip prompts.

## `contextual-translator`

Responsibilities:

- translate meaning rather than literally translating words,
- preserve names, ritual terms, and cultural nuance,
- adapt examples for Hindi/Telugu readers,
- maintain child-safe tone in kids content.

Output:

- Hindi story version,
- Telugu story version,
- localized FAQ/AEO answers,
- glossary notes.
